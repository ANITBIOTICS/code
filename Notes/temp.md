General:
    size() = size of the thing
    empty() = is it empty


Vector:
    push_back() = add at the end
    pop_back() = remove last element

Deque (double ended queue)
    push_back() = add at the end
    pop_back() = remove at the end
    push_front() = add at the front
    pop_front() = remove at the front

    front() = first element
    back() = last element
    clear() = removes all elements


Stack:
    first in last out
    push() = add to the list
    pop() = remove top of the list
    top() = get the top item
    empty() = true if the stack is empty, false otehrwise


Queue
    first in first out
